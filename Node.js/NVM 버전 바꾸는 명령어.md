현재 사용중인 Node.js 버전 확인

```bash
node -v 
nvm current
```

설치된 모든 Node.js 버전 목록 확인

```bash
nvm ls
```

사용 가능한 Node.js 버전 목록 확인

```bash
nvm ls-remote
```

특정 버전의 Node.js설치

```bash
nvm install [버전]
```

특정 버전의 Node.js 사용

```bash
nvm use [버전]
```

기본 Node.js 버전 설정

```bash
nvm alias default [버전]
```
