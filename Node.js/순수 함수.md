[[부수 효과]] 가 없는 함수입니다

### 1. 어떤 함수에 동일한 인자를 주었을 때 항상 같은 값을 리턴하는 함수

```js
let c = 10;

function add(a,b){
	return a + b + c;
}

c가 상수가 아니기 때문에 동일한 인자에 항상 같은 값을 리턴하지 않습니다. 따라서 순수 함수가 아닙니다
```
### 2. 외부의 상태를 변경하지 않는 함수

```js
let c = 10;

function add(a,b){
	c = b;
	return a + b ;
}

1번 조건은 만족하지만, 외부의 상태를 변경하고 있으므로 순수 함수가 아닙니다
```
